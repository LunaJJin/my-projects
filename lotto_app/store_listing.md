# Google Play Store 등록정보

## 앱 이름
행운의 로또번호

## 간단한 설명 (80자 이내)
2020~2026년 로또 당첨번호 통계 분석 및 빈도 기반 번호 추천 앱

## 자세한 설명 (4000자 이내)
로또 당첨번호 데이터를 분석하여 스마트하게 번호를 추천받으세요!

2020년부터 현재까지의 로또 6/45 당첨번호를 실시간으로 조회하고, 통계 데이터를 기반으로 번호를 추천받을 수 있는 앱입니다.

[주요 기능]

1. 당첨번호 이력 조회
- 2020년~현재까지 전체 회차 당첨번호 확인
- 회차별 당첨번호 + 보너스 번호 한눈에 보기
- 최신순 정렬로 빠르게 확인

2. 번호 통계 분석
- 가장 많이 나온 번호 TOP 10
- 가장 적게 나온 번호 TOP 10
- 구간별(1~10, 11~20, 21~30, 31~40, 41~45) 출현 비율 분석
- 직관적인 바 차트로 한눈에 파악

3. 스마트 번호 생성
- 과거 당첨번호 빈도 데이터를 기반으로 5세트 번호 자동 추천
- 통계적으로 의미 있는 번호 조합 생성
- 마음에 드는 번호는 바로 저장

4. 내 번호 관리
- 추천받은 번호 저장 및 관리
- 직접 번호를 선택하여 추가 가능
- 날짜별 번호 관리로 편리하게 확인

[특징]
- 실시간 데이터 업데이트
- 깔끔하고 직관적인 UI
- 오프라인에서도 캐시된 데이터 확인 가능
- 가볍고 빠른 앱 성능

※ 본 앱은 과거 데이터 기반 참고용이며, 로또 당첨을 보장하지 않습니다.
※ 로또 구매는 만 18세 이상만 가능합니다.
